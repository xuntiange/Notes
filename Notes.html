Use Cases List

  • Display a list of todo lis, which length is configurable
  • Todo lis in the shown list can be ordered by creation time, due time, priority, complexity
  • Press up arrow or down arrow to select any todo li, which will be highlighted
  • New todo li can be easily added through clipboard
  • New li and existed li can be added or edited in a separate window
  • Create li to remind me join a meeting
  • Create li to remind me to reply a mail
  • Store username and password of a newly created account
  • Record a technical problem or question, its solution or answer if available
  • Record a new idea or inspiration comes into mind
  • In general, it works like a notebook, but is more compact
